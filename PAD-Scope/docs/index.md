# PAD-Scope Documentation

**PAD-Scope** is a virtual oscilloscope through a logic analyzer on your board. It provides real-time visualization of digital and analog signals with advanced triggering and protocol decoding capabilities.

## Features

- Trigger modes for precise signal capture
- Protocol decoding for I2C/SPI/UART communications  
- Mathematical channels for signal processing
- Export data to CSV/WAV formats
- FFT analysis for frequency domain visualization

## Quick Start Guide

1. Connect your logic analyzer to the target device
2. Launch PAD-Scope
3. Configure sample rate and voltage thresholds
4. Set up triggers for specific events
5. Capture and analyze signals
