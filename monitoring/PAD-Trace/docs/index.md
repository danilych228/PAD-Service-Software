# PAD-Trace Documentation

**PAD-Trace** is a real-time power consumption profiler that correlates energy usage with code execution events. It helps optimize battery life and power efficiency in embedded applications.

## Features

- Real-time current/voltage sampling
- Sleep mode analysis and optimization
- Correlation of power events with code execution
- Power consumption reports ("mAh per task")
- Detailed analysis of power spikes and patterns

## Quick Start Guide

1. Connect the power monitoring hardware to your device
2. Launch PAD-Trace
3. Run your application while monitoring power consumption
4. Analyze correlation between code events and power usage
5. Optimize power-hungry sections of code
