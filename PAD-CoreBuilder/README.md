# PAD-CoreBuilder

PAD-CoreBuilder is a powerful generator for Arduino IDE/PlatformIO cores, enabling rapid development and customization of microcontroller support packages. With its intuitive interface, you can create custom cores for various MCUs with drag-and-drop peripheral configuration.

## Features

- Generate cores for Arduino IDE and PlatformIO
- Drag-and-drop peripheral configuration
- Export configurations to JSON format
- Preview register configurations
- Support for multiple MCU families
- Pin mapping utilities
- Peripheral template system
