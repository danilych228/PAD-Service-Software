# PAD-LogView Documentation

**PAD-LogView** is a log aggregator that collects logs from multiple devices over network/COM connections. It provides filtering, searching, and visualization capabilities for debugging distributed systems.

## Features

- Aggregation of logs from multiple devices simultaneously
- Advanced filtering and regex search capabilities
- Color-coded tags by log level (DEBUG, INFO, WARN, ERROR)
- Log rotation and archival
- Export to ELK stack for advanced analytics
- Session recording capabilities

## Quick Start Guide

1. Configure connection settings for your devices
2. Launch PAD-LogView
3. Connect to devices via network or COM ports
4. Apply filters and search for specific events
5. Export logs for further analysis
